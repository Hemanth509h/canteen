# Project Analysis & Price Estimation: Canteen Management System

## 1. Project Overview
This project is a comprehensive **Canteen and Event Catering Management System**. It manages the end-to-end workflow of a catering business, from menu management to staff assignments and payment tracking.

### Tech Stack
- **Frontend**: React (Vite), Tailwind CSS, Radix UI, TanStack Query.
- **Backend**: Node.js, Express, MongoDB (Mongoose), Redis (Sessions).
- **Integrations**: WhatsApp Business API, QR Code generation.

---

## 2. Feature Breakdown

### Core Modules
- **Menu Management**: CRUD for food items with categories and pricing.
- **Event Booking**: Track guest counts, per-plate pricing, and food selections.
- **Staff Management**: Assign roles (Serving Boys, Chefs) to specific bookings.
- **Financial Tracking**: Two-stage payment system (Advance/Final) with screenshot verification.
- **Chef Printout**: Daily summaries for kitchen staff.

### Advanced Features
- **WhatsApp Integration**: Automated payment links and notifications.
- **Secure Code System**: Lead qualification via requested booking codes.
- **Audit Logs**: Transparency for all financial and booking changes.
