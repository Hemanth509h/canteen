# Installation Commands

## Backend
```bash
cd backend
npm install
npm run dev
```

## Frontend
```bash
cd frontend
npm install
npm run dev
```

# Build Commands (Production)

## Backend
```bash
cd backend
npm run build
```

## Frontend
```bash
cd frontend
npm run build
```

# Environment Setup
Make sure you have `NODE_ENV` set up. The `cross-env` package handles this in the scripts provided.
